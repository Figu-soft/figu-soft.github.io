Política de Privacidad y Seguridad

Última actualización: 3 de febrero de 2026

PassGod es una herramienta diseñada para la generación de contraseñas seguras. Respetamos completamente la privacidad del usuario y hemos desarrollado esta aplicación para funcionar de manera totalmente offline, sin recopilar ni compartir información personal.

1. Información que recopilamos

PassGod no recopila, almacena ni transmite ningún tipo de información personal, sensible o identificable del usuario.

Todas las contraseñas generadas, así como los datos que el usuario decida guardar, permanecen exclusivamente en el dispositivo del usuario mediante una base de datos interna local.

2. Uso de la aplicación

PassGod permite:

  • Generar contraseñas mediante distintos parámetros configurables.
  • Guardar contraseñas generadas de forma local en el dispositivo.
  • Generar contraseñas a partir de archivos Excel proporcionados por el usuario.
  • Exportar o compartir archivos Excel generados por el propio usuario.

PassGod no es un gestor de contraseñas. La responsabilidad del uso, almacenamiento externo o compartición de las contraseñas generadas recae exclusivamente en el usuario.

3. Almacenamiento y seguridad

La aplicación utiliza una base de datos interna local del dispositivo para guardar las contraseñas que el usuario decide conservar. Estos datos:

  • No se sincronizan con la nube.
  • No se envían a servidores externos.
  • No son accesibles por terceros a través de PassGod.

4. Permisos de la aplicación

PassGod solo solicita los permisos estrictamente necesarios para su funcionamiento, como el acceso a archivos locales cuando el usuario decide importar o exportar archivos Excel.

La aplicación no accede a contactos, ubicación, cámara, micrófono ni a información sensible del sistema.

5. Aplicación de pago

PassGod es una aplicación de pago. No contiene servicios de seguimiento, y su modelo de negocio no depende de la recopilación de datos del usuario.

6. Privacidad de menores

PassGod puede ser utilizada por personas de todas las edades. Sin embargo, la aplicación no recopila ni procesa información personal de ningún tipo, incluidos datos de menores de edad.

7. Cambios en esta política

Esta política de privacidad y seguridad puede actualizarse ocasionalmente. Cualquier cambio será publicado en esta misma página.
